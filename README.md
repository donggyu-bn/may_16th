# Odelia & Donggyu Wedding Website

## Project Overview
A wedding website for Odelia and Donggyu's wedding on May 16th, 2026 at Europa Village.

## Theme: René Magritte Surrealism
The website is inspired by the surrealist paintings of René Magritte, featuring:
- Dreamy cloud-filled sky backgrounds reminiscent of Magritte's iconic imagery
- Surrealist visual elements and artistic imagery
- A decorative painting frame overlay creating the effect of viewing the website as a framed artwork
- The playful nod to Magritte's "Ceci n'est pas une pipe" with our title "Ceci n'est pas un mariage"
- Muted, painterly color palette with soft blues, creams, and earth tones

## Pages
- **Home** (`index.html`) - Main landing page with couple image, navigation buttons, and framed artwork aesthetic
- **Meet Odelia** (`odelia.html`) - Odelia's profile with polaroid-style childhood photos and cloud sky background
- **Meet Donggyu** (`donggyu.html`) - Donggyu's profile with matching layout
- **Our Story** (`story.html`) - How we met and our journey together
- **Letter** (`letter.html`) - Letter to guests with animated background
- **Before Letter** (`beforeletter.html`) - Pre-letter transition page
- **Menu** (`menu.html`) - Navigation hub linking to Venue, RSVP, Temecula, and Home
- **The Details / Venue** (`venue.html`) - Wedding ceremony and reception information
- **RSVP** (`rsvp.html`) - Guest response form
- **Temecula** (`temecula.html`) - Local area info about Temecula Valley wine country

## Navigation Flow
Home → Meet Odelia → Meet Donggyu → Our Story → Letter → Menu → (Venue / RSVP / Temecula)
